# Dynamic webpages with JavaScript

## Three Languages, Three Layers
- <HTML>
  - Content layer
- {css}
  - Presentation layer
- javascript()
  - Behavior layer

## Objects and Methods
- object: a collection of pre coded methods and properties that can be called to serve a purpose within code
  - the . after the object allows one to call a specific method within that object
  - when calling a method, include the name and a set of parentheses. within the parens you can include parameters like variables.
  
## Basic Java
Statements:
- Each step of code
- Ended with a semicolon
- Case sensitive
Comments:
  - Comments should be used frequently to keep track of what pieces of code do, both for you and anyone that may be reading it
Variables:
  - a piece of data saved by the program, either defined by you, an algorithm, user input, or a mix
  - descriptive names allow variables to be easily identifiable
  - declare a variable by stating it's type, then what you intend to name it

Data Types:
- Numeric
  - stores numbers
- String
  - stores phrases or combinations of characters
- Boolean
  - stores a value of true or false
  
  Rules for naming variables:
  1. cannot start with a number
  1. no dashes or periods
  1. no keywords or reserved words
  1. case sensitive
  1. be descriptive
  1. use camel case and/or underscores between words
